## 📌 Описание

**PasteTimer** — это desktop-приложение с графическим интерфейсом для автоматической отправки текста в активное окно с настраиваемым интервалом времени. Программа эмулирует набор текста и нажатие клавиши Enter, что позволяет автоматизировать отправку сообщений в любых мессенджерах, чатах или текстовых полях.

---

## ✨ Возможности

- 🖱️ **Автоматическая вставка текста** — программа сама печатает текст и нажимает Enter
- ⏱️ **Гибкая настройка интервала** — выбор между секундами и минутами (от 1 до 60)
- 🔄 **Многопоточная работа** — интерфейс не зависает во время отправки
- 📋 **Поддержка буфера обмена** — копирование и вставка внутри текстового поля (Ctrl+C, Ctrl+V)
- 🖱️ **Контекстное меню** — правый клик мыши для копирования/вставки
- 🎨 **Тёмная тема** — стильный интерфейс в тёмных тонах
- 🚦 **Статус работы** — визуальный индикатор активного процесса
- 🛑 **Управление** — кнопки Старт/Стоп для полного контроля

## 🎮 Как пользоваться

1. **Введите текст** в текстовое поле (можно вручную или вставить через Ctrl+V)
2. **Выберите режим** интервала: Секунды или Минуты
3. **Настройте интервал** с помощью ползунка (от 1 до 60)
4. **Нажмите «СТАРТ»** — программа начнёт отправку
5. **Переключитесь в нужное окно** (мессенджер, чат, поле ввода)
6. **Для остановки** нажмите «СТОП»

---

## ⌨️ Горячие клавиши

| Сочетание | Действие |
|-----------|---------|
| `Ctrl+C` | Копировать выделенный текст |
| `Ctrl+V` | Вставить текст из буфера |
| `Ctrl+X` | Вырезать выделенный текст |

## ⚠️ Примечания

- Программа работает в активном окне — убедитесь, что курсор находится в нужном поле ввода
- Для работы в некоторых играх или защищённых приложениях может потребоваться запуск от имени администратора
- Текст отправляется как единое сообщение (переносы строк заменяются пробелами)
